
magic powers
============

:: Description
A character is able to do at will some fantastical things that can only be
described as "magic".

:: Parents
supernatural ability

:: Notes
This theme is used when the story itself implies magic, rather than for
example speculative technology or other categorized speculative ability.



mind control ability
====================

:: Description
A character, by simply using their mind, is able to control other people's
thoughts and actions.

:: Parents
supernatural ability

:: References
https://tvtropes.org/pmwiki/pmwiki.php/Main/MindManipulation



miracle working
===============

:: Description
There is a character that is able to work miracles at will.

:: Parents
supernatural ability

:: Notes
Miracle working typically has a connection with religion and deities. It is
generally not used to describe situations in which a character is able to
immediately manipulate the world with his mind or through spells etc. In
particular, telekinetics, fire conjuring sourcery, and suchlike wizardry is
not miracle working. This may be a matter of interpretation in some cases.

:: Examples
In nightgallery1x04a "Make Me Laugh", a miracle working Indian swami helps
Jackie Slater by making it so that he effortlessly inspires laughter in anyone
who listens to him.

:: References
https://en.wikipedia.org/wiki/Thaumaturgy
https://en.wikipedia.org/wiki/Miracle

=====================

:: Description

themself, others, or objects, to remote locations, as by magic.

:: Parents

:: Examples

to use will alone to move himself in space. Thus he made good his escape, and
